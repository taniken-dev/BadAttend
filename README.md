# バドミントン部 出欠管理アプリ (BadAttend)

バドミントン部の活動を効率化する出欠管理プラットフォームです。

## 開発の背景

チャットベースでの出欠連絡が煩雑化しており、直前の欠席連絡が常態化していた問題を解決するために開発しました。

## 主な機能

### 出欠管理
- **出欠連絡:** カレンダーから各練習セッションの出欠を登録（出席・遅刻・欠席・当日欠席）
- **欠席理由の記録:** 別練習・授業・体調不良・私用・その他から選択
- **遅刻理由の必須入力:** 遅刻時は理由が必須。授業の場合は何限かを選択（8〜10限・終了時刻付き）
- **遅刻の参加予定時刻選択:** 遅刻登録時に参加予定時刻を選択できる
- **当日欠席・当日遅刻:** 練習当日に欠席/遅刻を登録すると「当日欠席」として記録され、LINEグループへ自動通知
- **事前欠席連絡:** 登録期間外・当日前でも欠席へ変更可能。LINEグループへ「事前連絡」として区別して通知
- **当日登録ウィンドウ:** 練習開始後も当日中は出欠登録が可能
- **体調不良ロック:** 体調不良で欠席した場合、次回練習への参加を制限し安易な嘘を抑止
- **実績確定:** 管理者がセッションごとに出欠実績を確定。確定済みデータのみスコアに反映

### セッションの種類
- **通常練習:** 出欠連絡・実績確定・スコア算出の対象
- **部会:** カレンダー上で区別して表示。出席率・選考スコアの計算からは除外
- **合宿:** 特別セッションとして管理。出席率・選考スコアの計算からは除外
- **自主練習:** 参加は任意。参加意思表示と到着予定時刻の選択・変更・取り消しができる。週次リマインドの対象外
- **複数セッション/日:** 同じ日に部会＋部活など複数のセッションを持てる

### ホーム (ダッシュボード)
- 今日の練習の参加予定者・欠席連絡者をリアルタイム表示
- 参加予定者を時間帯ごとにグループ表示（遅刻者は参加予定時刻ごとにまとめて表示）
- 自分の出席率・出席/遅刻/欠席の内訳・直近10回の活動ドット表示
- 体調不良ロック中の場合はバナー通知

### カレンダー
- 月表示のたびにGoogleカレンダーと自動同期し練習スケジュールをDB管理
- コート面数・補足時間（時間注記）の表示に対応
- 出欠リスト・未提出リストの名前検索絞り込み
- 出欠リストを出欠状況順にデフォルトソート（複数キーによる並び替え対応）
- 未提出者へのLINE個別リマインド通知（管理者操作）
- **練習休止機能:** 管理者が休止理由（例：台風のため）とともにセッションを休止設定可能。休止日は出欠登録・実績登録・LINEリマインドを全てブロック（出席率計算には影響なし）

### LINE通知
- **当日欠席/遅刻の自動グループ通知:** 当日に欠席・遅刻を登録すると、理由を含む通知がLINEグループへ即座に送信される
- **事前欠席の自動グループ通知:** 登録期間外に欠席へ変更すると「事前連絡」として通知される（当日欠席と文面を区別）
- **未提出者への個別リマインド:** 管理者が「LINEでリマインド」ボタンを押すと未提出メンバー全員にマルチキャスト送信
- **週次自動リマインド（Vercel Cron）:** 毎週火曜11:00に未提出メンバーへ自動でLINE通知（自主練習セッションは対象外）

### ランキング (選考スコア)
- 出席率と技術ランク（E〜S級）を掛け合わせた独自の選考スコアを表示
- 全体・学年別のタブ切替
- 出席回数6回未満のフィルター
- 確定済み実績ベースで算出（合宿・部会は対象外）
- 各部員の入部日以降のセッションのみを集計（入部日未設定の場合は学年から推定）

### 意見箱
- 部員が匿名で意見・要望を投稿できる（投稿は承認済み部員のみ、タイトル100文字・本文1000文字まで）
- 管理者は管理画面から投稿を確認できる

### 管理者向け管理機能
- **メンバー管理:** 入部申請の承認・却下、学年/ロール/技術ランク/入部日の編集、LINEアイコン表示、孤立ユーザー検出、名前検索
- **退部処理（ソフトデリート）:** 退部処理をしても出席記録等のデータは保持され、退部済みメンバーとして管理画面から確認できる
- **実績管理:** セッションごとの実績確定・個別確定・取消・全解除
- **注意勧告フラグ:** 無連絡欠席・会費未納・行動に対する警告管理（警告/最終警告/除籍）

### 提出書類の締切管理（体育会・幹部向け）
- 体育会本部HP（Googleサイト）の「提出期限間近の書類」を毎日自動取得（Vercel Cron）し、ホーム画面にカード表示
- 表示されるのは管理者と「幹部」フラグが設定されたユーザーのみ
- 各書類に「提出しました」チェックが付けられ、誰がいつチェックしたかを記録・表示
- 未提出の書類は締切の7・3・1日前と当日に管理者・幹部へLINEで自動リマインド（提出済みは通知対象外）
- 取得失敗やHP書式変更の疑い（取得0件）を検知すると管理者・幹部へLINEで異常通知
- カードに最終取得日時と体育会HPへのリンクを常時表示

### 出欠ガイド
- `/rules` ページで登録期間・ステータスの種類・遅刻/欠席の理由・登録手順を確認できる

### 利用規約・プライバシーポリシー
- `/terms` に利用規約ページ（全8条）
- `/privacy` にプライバシーポリシーページ（全8項）
- ログイン画面に両ページへのリンクと同意文を表示

### ロール体系

| ロール | 説明 |
|--------|------|
| `member` | 一般部員 |
| `manager` | マネージャー（技術ランクなし） |
| `admin` | 管理者（全管理機能にアクセス可） |
| `coach` | 顧問（個人スコア非表示、閲覧専用） |

4ロールとは別に、ロールと直交する「**幹部**」フラグ（`is_executive`）がある。設定できるのは管理者のみ（メンバー管理画面のチェックボックス）。幹部または管理者にのみ体育会の提出書類締切が表示される。

## 使用技術 (Tech Stack)

- **Frontend:** Next.js (App Router) / Tailwind CSS
- **Backend:** Supabase (Auth, Database, RLS)
- **Auth:** LINE Login (OAuth) のみ
- **外部連携:** Google Calendar API / LINE Messaging API
- **デプロイ:** Vercel（Cron Jobs 使用）

## 運営ルール
- 出欠連絡はその週の土曜日〜火曜日23:59までに提出（水・木・金の練習対象）
- 水・木・金の当日は深夜0時まで登録可能
- 登録期間外でも欠席への変更は事前欠席連絡として受け付ける
- 体調不良で欠席した場合は体調不良ロックが適用される
- 当日の欠席・遅刻、および事前欠席変更はLINEグループへ自動通知される
- 練習休止日は出欠登録・実績登録・LINEリマインドが無効化される
- 合宿・部会は出席率・選考スコアの計算に含まれない
- 自主練習は任意参加で、出欠連絡・週次リマインドの対象外
