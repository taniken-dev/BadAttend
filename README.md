# バドミントン部 出欠管理アプリ (BadAttend)

バドミントン部の活動を効率化する出欠管理プラットフォームです。

## 開発の背景

チャットベースでの出欠連絡が形骸化しており、直前の欠席連絡が常態化していた問題を解決するために開発しました。部則にある「競技力より練習時の取り組み姿勢を重視する」という方針をシステム化し、公平な選手選考とチームの規律強化を目的としています。

## 主な機能

### 出欠管理
- **出欠連絡:** カレンダーから各練習セッションの出欠を登録（出席・遅刻・欠席・当日欠席）
- **欠席理由の記録:** 別練習・授業・体調不良・私用・その他から選択
- **遅刻理由の必須入力:** 遅刻時は理由が必須。授業の場合は何限かを選択（8〜10限・終了時刻付き）
- **当日欠席・当日遅刻:** 練習当日に欠席/遅刻を登録すると「当日欠席」として記録され、LINEグループへ自動通知
- **当日登録ウィンドウ:** 練習開始後も当日中は出欠登録が可能
- **体調不良ロック:** 体調不良で欠席した場合、次回練習への参加を制限し安易な嘘を抑止
- **実績確定:** 幹部がセッションごとに出欠実績を確定。確定済みデータのみスコアに反映

### ホーム (ダッシュボード)
- 今日の練習の参加予定者・欠席連絡者をリアルタイム表示
- 自分の出席率・出席/遅刻/欠席の内訳・直近10回の活動ドット表示
- 体調不良ロック中の場合はバナー通知

### カレンダー
- Googleカレンダーと自動同期し練習スケジュールをDB管理
- 合宿などの特別セッション対応
- 出欠リスト・未提出リストの名前検索絞り込み
- 未提出者へのLINE個別リマインド通知（幹部操作）

### LINE通知
- **当日欠席/遅刻の自動グループ通知:** 当日に欠席・遅刻を登録すると、理由を含む通知がLINEグループへ即座に送信される
- **未提出者への個別リマインド:** 幹部が「LINEでリマインド」ボタンを押すと未提出メンバー全員にマルチキャスト送信
- **週次自動リマインド（Vercel Cron）:** 毎週火曜11:00に未提出メンバーへ自動でLINE通知

### ランキング (選考スコア)
- 出席率と技術ランク（E〜S級）を掛け合わせた独自の選考スコアを表示
- 全体・学年別のタブ切替
- 出席回数6回未満のフィルター
- 確定済み実績ベースで算出

### 幹部向け管理機能
- **メンバー管理:** 入部申請の承認・却下、学年/ロール/技術ランクの編集、LINEアイコン表示、孤立ユーザー検出、名前検索
- **実績管理:** セッションごとの実績確定・個別確定・取消・全解除
- **意見箱:** 部員からの匿名の意見・要望を確認
- **注意勧告フラグ:** 無連絡欠席・会費未納・行動に対する警告管理（警告/最終警告/除籍）

### 出欠ガイド
- `/rules` ページで登録期間・ステータスの種類・遅刻/欠席の理由・登録手順を確認できる

### ロール体系

| ロール | 説明 |
|--------|------|
| `member` | 一般部員 |
| `manager` | マネージャー（技術ランクなし） |
| `admin` | 幹部（全管理機能にアクセス可） |
| `coach` | 顧問（個人スコア非表示、閲覧専用） |

## 使用技術 (Tech Stack)

- **Frontend:** Next.js (App Router) / Tailwind CSS
- **Backend:** Supabase (Auth, Database, RLS)
- **Auth:** LINE Login (OAuth) のみ
- **外部連携:** Google Calendar API / LINE Messaging API
- **デプロイ:** Vercel（Cron Jobs 使用）

## 環境変数

| 変数名 | 用途 |
|--------|------|
| `NEXT_PUBLIC_SUPABASE_URL` | Supabase プロジェクト URL |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | Supabase 匿名キー |
| `SUPABASE_SERVICE_ROLE_KEY` | Supabase サービスロールキー |
| `LINE_MESSAGING_CHANNEL_ACCESS_TOKEN` | LINE Messaging API アクセストークン |
| `LINE_GROUP_ID` | 通知送信先のLINEグループID |
| `CRON_SECRET` | Vercel Cron 認証トークン |

## 運営ルール
- 出欠連絡はその週の土曜日〜火曜日23:59までに提出（水・木・金の練習対象）
- 水・木・金の当日は深夜0時まで登録可能
- 体調不良で欠席した場合は体調不良ロックが適用される
- 当日の欠席・遅刻はLINEグループへ自動通知される
